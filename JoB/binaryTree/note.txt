คืองี้ทีแรก input เข้าไปเพื่อสร้าง node ทีนี้มันก็จะสร้าง Tree

แล้ว print แแต่ละขั้นตอนในการสร้าง Tree ขึ้นมา

พอเสร็จแล้วมันก็จะเรียกใช้ function sort แล้ว print ค่าจาก Tree ออกมาเรียงจากน้อยไปมาก

แล้วทีนี้ก็ใส่ค่า node ที่จะลบ ไป

มันก็จะทำการลบ node ที่ใส่ไปแล้ว print ขั้นตอนการลบออกมา
